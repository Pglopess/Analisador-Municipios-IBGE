# Analisador de Municípios IBGE

Script Python que lê dados públicos de municípios brasileiros, processa as informações e gera um relatório com estatísticas por região.

## Tecnologias
- Python 3.9+
- Módulo `csv` (biblioteca padrão)

## Como executar
```bash
python main.py
```

## Estrutura do projeto
